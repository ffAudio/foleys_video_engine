Detailed Description

This class sends a single, anonymous ping to our analytics collection.

To use foleys_video_engine using the free "Personal License", you must keep this in your code and avoid anything, that might render this data collection ineffective.

If you want to opt-out, you will have to buy one of the paid licenses.
